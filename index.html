Fuzzy Decision Support System (DSS) User Guide & Operator Manual

Welcome to the Fuzzy Multi-Criteria Decision Support System (MCDSS). This system employs Mamdani Fuzzy Logic Inference to evaluate complex qualitative metrics (such as talent fit, soft skills, or resource parameters) under real-world, uncertain constraints.

1. Define Requirements (Box A)

Paste a job description text, load standard templates, or paste a Naukri/LinkedIn job link. The system extracts core skills and requirements in real-time.

2. Bulk Screen Resumes (Box B)

Drag and drop multiple text resumes (.txt) or click to upload. The system batch-parses all files, matching skills and calculating cosine similarities instantly.

3. Analyze Leaderboard

The sortable Leaderboard Table ranks candidates by their defuzzified compatibility score. Click any row to load detail charts, a resume viewer, and natural summaries.

4. Priority Likert Matrix

Switch to the Decision Nexus tab to evaluate links and places. Score Importance, Urgency, and Necessity on a 1-5 scale to dynamically calibrate rules.
